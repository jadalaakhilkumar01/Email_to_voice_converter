# Email to Voice Converter 🔊

A browser extension that converts email content into speech using Text-to-Speech (TTS) technology.

##  Features
- Converts email text into audio
- Works directly inside the browser
- Improves accessibility and convenience
- One-click voice playback

##  Tech Stack
- JavaScript
- HTML, CSS
- Web Speech API (TTS)

##  Project Structure
- content.js → extracts email text
- popup.js → handles UI interaction
- popup.html → extension interface
- manifest.json → extension configuration

##  How to Use
1. Open Chrome → Extensions → Enable Developer Mode
2. Click "Load unpacked"
3. Select this project folder
4. Open email → Click extension → Listen to audio

##  Use Case
Helps users listen to emails instead of reading them.
